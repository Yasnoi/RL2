### Rollout代码研读总结

#### 核心架构
- **Rollout类**：负责任务编排和推理服务器管理
- **SampleGroup**：一个prompt的多个response（通过responses_per_prompt控制）
- **Sample**：单个trajectory，包含state、action、reward序列

#### 关键流程
1. 初始化：启动SGLang推理服务器 + Router
2. 推理循环（`__call__`）：
   - rank 0 调度asyncio任务并发生成
   - 每个Sample：LLM生成 → env.step() → 获取reward → 判断是否结束
3. 权重更新（`update`）：训练后将权重同步到推理服务器
